## 2.1 自回避随机行走问题

### 问题描述

给定一个 N x N 的网格地图, 初始在地图中心.

每次随机向**四个相邻位置中还没有走过的点**移动, 如果无路可走则陷入死胡同.

编程**模拟**以上过程, 进行多轮尝试, 分析:

1. 陷入死胡同的概率
2. 行走的平均长度
3. 成功逃出时的平均路径长度
4. 陷入死胡同时的平均路径长度
5. N与陷入死胡同的概率的关系

### 核心函数

随机行走, 传入引用以实现多个返回值, ok表示是否成功逃出, length表示路径长度

```C++
void randomWalk(int N, bool &ok, int &length);
```

### Random类

用于获取随机数, 对rand()和srand()的一个封装.
